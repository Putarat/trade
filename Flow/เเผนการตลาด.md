# แผนการตลาด — Trade Portfolio Tracker

> เป้าหมาย: ดึงนักลงทุนไทยให้เข้ามาใช้งานและบอกต่อ

---

## 1. กลุ่มเป้าหมาย (Target Audience)

### Primary
| กลุ่ม | ลักษณะ | ขนาดตลาด |
|---|---|---|
| นักลงทุนหน้าใหม่ | อายุ 20–35 ปี เพิ่งเริ่มลงทุน ยังไม่มีเครื่องมือ track พอร์ต | ~500,000 คน |
| นักลงทุนรายย่อย SET | ใช้ Settrade/Jitta อยู่แล้ว แต่อยากได้ตัว track แบบ all-in-one | ~1.2M คน |
| นักลงทุน US + Thai | ลงทุนทั้ง 2 ตลาด ต้องการดู P&L รวมเป็น THB | ~150,000 คน |

### Secondary
- กลุ่ม FIRE / ออมเพื่อเกษียณ
- เทรดเดอร์ที่ต้องการ track หลายพอร์ต

## 2. จุดแตกต่าง (Unique Value Proposition)

> **"จัดการพอร์ตหุ้นไทย + US ในที่เดียว เห็น P&L จริง เป็นบาท"**

| คู่แข่ง | ข้อจำกัด | เราดีกว่าอย่างไร |
|---|---|---|
| Settrade | ดูได้แค่หุ้นไทย | รองรับทั้ง SET + NYSE/NASDAQ |
| Jitta | เน้น screener ไม่ใช่ portfolio tracker | Track qty/cost/P&L ได้ละเอียด |
| Google Sheets | ต้องกรอกเอง สูตรซับซ้อน | Scan screenshot ได้เลย + auto price |
| Portfolio apps ต่างประเทศ | ไม่รองรับ SET / ราคาเป็น THB | ออกแบบมาสำหรับนักลงทุนไทยโดยตรง |

## 3. ช่องทางการตลาด (Channels)

### 3.1 Social Media (เน้นก่อน)

**Facebook Groups** ⭐ สำคัญที่สุด
- กลุ่ม "หุ้น SET", "VI Thailand", "ลงทุนหุ้น US"
- โพสต์รีวิวตัวเอง / ให้สมาชิกในกลุ่มทดลองใช้ฟรี
- อย่า spam — ให้ value ก่อน แล้ว mention แอป

**TikTok / YouTube Shorts**
- คลิปสั้น "วิธี track พอร์ตหุ้น 5 นาที"
- "กำไร/ขาดทุนจริงหรือเปล่า? คำนวณยังไง"
- ใช้ฟีเจอร์ Scan เป็น hook ดึงดูด

**X (Twitter)**
- เจาะกลุ่ม $NVDA $AAPL ไทย + #หุ้นไทย
- แชร์ insight จาก Mega Trend Panel ที่แอปทำได้

**Pantip**
- บอร์ด หุ้น / การเงิน
- รีวิวจากผู้ใช้จริง (ไม่ใช่การตลาดตรง)

---

### 3.2 Content Marketing

**บทความ SEO (ยาวกว่า 6 เดือน)**

| หัวข้อ | Keyword เป้า |
|---|---|
| วิธี track พอร์ตหุ้นไทย + US | "track พอร์ตหุ้น" |
| คำนวณ P&L หุ้นยังไงให้ถูก | "คำนวณกำไรหุ้น" |
| เมกะเทรนด์ 2025 ลงทุนอะไรดี | "mega trend หุ้น 2025" |
| Avg cost คืออะไร ทำไมสำคัญ | "average cost หุ้น" |

**Newsletter / Line OA**
- ส่ง market update ทุกอาทิตย์
- แนบ insight จากแอป เช่น "สัปดาห์นี้ Mega Trend AI ผันผวน x%"

---

### 3.3 Community & Referral

- **Beta Tester Program** — รับ 50 คนแรก ใช้ฟรีตลอด + ขอ feedback
- **Referral** — เชิญเพื่อน 1 คน ได้ scan credit ฟรี 5 ครั้ง
- **Review บน Pantip / Facebook** — ขอรีวิวจากผู้ใช้จริง ให้ credit เป็นรางวัล

## 4. Pricing & Monetization

### แผนปัจจุบัน (ใน App)
| แผน | ราคา | Credits |
|---|---|---|
| สแกน 1 ครั้ง | ฿20 | 1 |
| แพ็กเริ่มต้น | ฿149 | 10 |
| รายเดือน | ฿299 | 30/เดือน |

### แนะนำ: Freemium Model
```
Free tier:
  - Track ได้ไม่จำกัดหุ้น
  - ดู P&L / Watchlist / เมกะเทรนด์
  - 3 scan ฟรีต่อเดือน

Pro (฿99/เดือน):
  - Scan ไม่จำกัด
  - หลายพอร์ต (ปัจจุบัน Create New Portfolio)
  - Export PDF / CSV
  - Price alert
```

> ข้อได้เปรียบ: Free tier ดึงคนเข้ามา → ใช้จนติด → Convert เป็น Pro

## 5. Growth Hacking

### Phase 1 — Launch (เดือน 1–2)
- [ ] โพสต์ใน 5 Facebook Group หุ้นไทย ขอ beta tester
- [ ] TikTok 2 คลิป/สัปดาห์ สอน track พอร์ต
- [ ] ให้ 100 คนแรกใช้ฟรี 3 เดือน แลกรีวิว
- [ ] ตั้ง Line OA / Discord สำหรับ community

### Phase 2 — Growth (เดือน 3–4)
- [ ] เปิด Referral Program อย่างเป็นทางการ
- [ ] ส่ง cold email ถึง influencer หุ้นไทย (YouTube < 50K sub)
- [ ] บทความ SEO 4 บทความ
- [ ] Google Ads ทดสอบงบ ฿5,000/เดือน

### Phase 3 — Scale (เดือน 5–6)
- [ ] Partnership กับ broker ไทย (อาจ co-promote)
- [ ] Product Hunt launch (English version)
- [ ] Press release ใน The Standard / Finnomena
